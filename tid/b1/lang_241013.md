# TCL语言记录

## 语法

两种变量表示法：双引号和花括号。单行双引号支持$替换，而花括号则用于跨号

```
set a "val $num"
# if命令接受两个参数，所以用花括号包住
if {a > 0} {
  puts "hi"
}
```

## TK与TTK

ttk是TK8.5时将原来的三方包合入主流。有几个基础概念：窗口、容器、布局、组件widget

绘制顺序是先从tk构建root窗口，然后构造容器或组件并往窗口布局

### 组件容器

* Frame/LabelFrame 子窗口/带标签子窗口
* PanedWindow 并排容器
* Canvas 画布
* Toplevel 
* ttk.Notebook、Treeview: ttk特有

### 组件

* Entry 单行输入
* Text 多行输入
* Label 静态文本

### 布局

继承顺序，Grid->Place->Pack

常用法：root用grid布局放置Frame，然后Frame内部用pack填充功能组件。
